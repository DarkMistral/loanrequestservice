# Loan Request Service

## About

The LoanRequestService is a prototypical REST service, built with Java and Spring Boot.
It's also using an in-memory H2-database, which will be built up and teared down with your program startup/shutdown.


## Start the application

To run the service locally, you can use the maven wrapper to get the necessary dependencies and either your IDE or use the `mvn spring-boot:run` command in your command line.
Since I've kept this basic, there are no different profiles or configurations.
